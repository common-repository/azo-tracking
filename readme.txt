AZO Tracking is a comprehensive plugin that integrates Google Analytics (GA4) into your WordPress website. It shows analytics data with elegant UI/UX.

== Description ==

AZO Tracking is a powerful tool for managing Google Analytics data in WordPress website. It offers a variety of features to track and display your website's analytics in a user-friendly interface.

[Home](https://azonow.com/)  · [Help & Support](https://my.azonow.com/support/) · 

= Features =
* Integrate GA4 into your website.
* Display Google Analytics statistics with a sleek UI/UX.
* Track scroll depth events.
* Monitor error events.
* Host 'gtag.js' locally.
* Configure GA4 script attributes.
* Display a custom dashboard for specific user roles.
* Create a customized statistics dashboard.

= Google Analytics Statics  =
* General stats
* Daily visitors stats
* Visitor devices stats
* New vs. returning visitors stats
* Operating systems stats
* Referrer stats
* Browser stats
* Top pages stats
* Current activity stats
* Geographic stats

== Installation ==

How to install the plugin and get it working?

= Using The WordPress Dashboard (Recommended Installation) =

1. Navigate to the 'Add New' in the Plugins dashboard.
2. Search for 'AZO Tracking'.
3. Click 'Install Now'.
4. Activate AZO Tracking on the Plugin dashboard.

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the Plugin dashboard.
2. Navigate to the 'Upload' area.
3. Select `azo-tracking.zip` from your computer.
4. Click 'Install Now'.
5. Activate AZO Tracking in the Plugin dashboard.

= Using FTP =

1. Download `azo-tracking.zip`
2. Extract the `azo-tracking` directory to your computer
3. Upload the `azo-tracking` directory to the `/wp-content/plugins/` directory
4. Activate AZO Tracking in the Plugin dashboard
